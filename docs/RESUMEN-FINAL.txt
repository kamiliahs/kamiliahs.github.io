❌ ERRORES COMUNES Y SOLUCIONES:

  Error: "Permiso de cámara denegado"
  → Habilitar en configuración del navegador
  → Fallback: Ingresar IP manualmente

  Error: "No hay cámara disponible"
  → Usar entrada manual (IP: 192.168.1.x | ID: peer_...)
  → O usar otro dispositivo con cámara

  Error: "QR no se detecta"
  → Verificar iluminación (bien iluminado)
  → Verificar distancia (15-30 cm)
  → Limpiar lente de cámara
  → Fallback: Entrada manual

  Error: "No conecta"
  → Verificar mismo WiFi
  → Verificar servidor activo
  → Ingresar IP correcta manualmente
