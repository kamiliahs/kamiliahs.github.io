# Carpeta de Recursos

Aquí puedes organizar:

- **images/** - Imágenes PNG, JPG, SVG
- **fonts/** - Fuentes personalizadas (TTF, WOFF, etc.)
- **icons/** - Iconos SVG
- **videos/** - Videos embebidos

## Ejemplo de estructura

```
assets/
├── images/
│   ├── logo.svg
│   ├── hero-background.jpg
│   └── portfolio/
├── fonts/
│   └── custom-font.woff2
├── icons/
│   ├── github.svg
│   ├── linkedin.svg
│   └── twitter.svg
└── videos/
    └── demo.mp4
```

## Optimización de imágenes

Para mejor rendimiento en GitHub Pages:

- Comprime imágenes con [TinyPNG](https://tinypng.com/)
- Usa formatos modernos: WebP cuando sea posible
- Optimiza el tamaño: mantén imágenes bajo 500KB
