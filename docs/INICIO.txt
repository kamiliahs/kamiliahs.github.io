🚀 CÓMO USAR:

  COMO SERVIDOR:
  ────────────
  1. Abre la app
  2. "Usar Red Local" → SÍ
  3. "Selecciona tu rol" → SERVIDOR
  4. ¡QR generado automáticamente!
  5. Muestra el código a los clientes

  COMO CLIENTE:
  ────────────
  1. Abre la app
  2. "Usar Red Local" → SÍ
  3. "Selecciona tu rol" → CLIENTE
  4. "Escanear QR"
  5. Apunta a la cámara hacia el código
  6. ¡Conectado automáticamente!

⚡ CARACTERÍSTICAS TÉCNICAS:

  • Tiempo de detección: <3 segundos
  • Latencia de sincronización: <1 segundo
  • Soporte navegadores: Chrome, Firefox, Safari, Edge (moderno)
  • Requisitos: Mismo WiFi, cámara opcional
  • Datos: Almacenados localmente (sin nube)
  • Comunicación: P2P vía localStorage
  • Seguridad: QR válido solo esa sesión

❌ POSIBLES PROBLEMAS Y SOLUCIONES:

  Problema: "Permiso de cámara denegado"
  ├─ ✓ Solución: Habilitar en configuración del navegador
  └─ ✓ Fallback: Ingresar IP manualmente

  Problema: "No hay cámara disponible"
  ├─ ✓ Solución: Usar entrada manual
  └─ ✓ IP: Ej. 192.168.1.50 | ID: peer_1704...

  Problema: "QR no se detecta"
  ├─ ✓ Verifica iluminación (bien iluminado)
  ├─ ✓ Verifica distancia (15-30 cm)
  ├─ ✓ Verifica que sea el QR correcto
  └─ ✓ Fallback: Entrada manual
