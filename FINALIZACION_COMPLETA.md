## 💡 Tips Importantes

### Para Mantener las Optimizaciones
- Siempre usar **variables CSS** en lugar de valores hardcoded
- Siempre agregar **data-i18n** a nuevos textos
- Siempre respetar **touch targets 44x44px**
- Siempre usar **event delegation** en JavaScript
- Siempre probar en **375px de ancho**

### Agregar Nuevo Contenido
```
1. HTML → Crear elemento semántico
2. CSS → Usar variables var(--...)
3. JavaScript → Event delegation
4. i18n → Traducir a ES/EN
5. Test → Probar en 375px + keyboard + screen reader
```

### Testing Rápido
```
Chrome DevTools:
- F12 → Device Mode → 375px
- F12 → Lighthouse
- F12 → Console (sin errors)
- F12 → Network (SW caching)
- Tab key → Keyboard navigation
```
